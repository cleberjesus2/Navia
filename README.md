Navia: Conectando Pessoas à Sustentabilidade e ao Consumo Consciente

Bem-vindo ao Navia! Um aplicativo projetado para promover eventos sustentáveis e facilitar o consumo consciente por meio de um marketplace de produtos ecológicos e recicláveis. O Navia une praticidade e responsabilidade ambiental, oferecendo uma plataforma intuitiva e acessível para todos.

🌍 Sobre o Projeto

O Navia nasceu com o propósito de criar um impacto positivo no mundo, incentivando práticas sustentáveis e conectando pessoas interessadas em um estilo de vida mais consciente. Seja participando de eventos ecológicos, comprando produtos recicláveis ou aprendendo novas dicas para um cotidiano mais sustentável, o Navia oferece tudo em um só lugar.

🎯 Objetivo

Nosso objetivo é capacitar indivíduos e organizações a adotarem hábitos sustentáveis, promovendo a preservação ambiental e criando uma rede colaborativa para compartilhar eventos, produtos e ideias. Queremos tornar a sustentabilidade mais acessível, prática e inclusiva para todos.

🏆 Funcionalidades

🌱 Para Usuários

Exploração de Eventos: Descubra eventos sustentáveis na sua região ou em qualquer lugar, com descrições detalhadas e opções de inscrição.

Marketplace Verde: Encontre e adquira produtos ecológicos, recicláveis e reutilizáveis, diretamente na plataforma.

Perfil e Histórico: Acompanhe seus eventos e compras realizadas, visualize favoritos e gerencie suas configurações.

Dicas Ecológicas: Receba orientações práticas para reduzir desperdícios, economizar recursos e adotar hábitos conscientes.


💡 Para Anunciantes

Criação de Eventos: Publique eventos voltados à sustentabilidade e divulgue para uma comunidade engajada.

Venda de Produtos: Anuncie itens ecológicos e conecte-se com consumidores interessados no consumo responsável.


🔔 Funcionalidades Extras

Notificações Personalizadas: Alertas sobre eventos, produtos em destaque, e novidades que atendem ao seu perfil.

Seja um Segurança: Ferramenta para organizadores recrutarem seguranças e garantirem a tranquilidade dos eventos.

Recomendações Baseadas no Perfil: Sugestões de eventos e produtos alinhados aos seus interesses.

Rede de Sustentabilidade: Colabore com outros usuários compartilhando ideias e práticas ecológicas.


📱 Tecnologias Utilizadas

O Navia foi desenvolvido utilizando ferramentas modernas para oferecer um ambiente seguro e eficiente:

Front-end: Ionic para um design responsivo e de fácil navegação em dispositivos móveis.

Back-end: PHP para processamento de dados, com integração ao banco de dados.

Banco de Dados: MySQL para armazenamento e gerenciamento de informações com segurança e confiabilidade.

⚙️ Como Funciona

1. Cadastro/Autenticação: Crie sua conta com dados pessoais e inicie sua jornada sustentável.


2. Exploração: Navegue pelas categorias de eventos e produtos disponíveis.


3. Ação: Inscreva-se em eventos, adquira produtos ou adicione favoritos.


4. Colaboração: Anuncie seus próprios eventos e produtos ecológicos.



🚀 Começando

Pré-requisitos

Instale o Ionic CLI para rodar o app localmente.

Configure o banco de dados MySQL para uso com o back-end PHP.


Instalação

1. Clone o repositório:

git clone https://github.com/seu-usuario/Navia.git


2. Entre na pasta do projeto:

cd Navia


3. Instale as dependências:

npm install


4. Configure a conexão com o banco de dados no arquivo db.php.


5. Inicie o aplicativo:

ionic serve
